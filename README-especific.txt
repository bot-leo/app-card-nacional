# Card-Nacional APP

- Objetivo :
    - Aplicativo voltado para o usuário (Contratante do produto)
    - Informar e mostrar o plano contratado
    - Listar produtos disponíveis
    - Realizar a mudança de Plano
    - Fazer pagamentos pelo App
    - Informar o carro cadastrado no plano
    - Poder alterar o carro cadastrado no plano
    - Informar serviços(quantidade) que ja foram utilizado no plano
    - Poder ter renovação de contrato automático
    - Armazenar informações do usuário
    - Permitir o usuário alterar informações pessoais
    - Solicitar assistência pelo app
